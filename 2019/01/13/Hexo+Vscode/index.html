实现hexo new 自动打开vscode编辑生成的md文档

  • 安装 Visual Studio Code

  • 前往 Visual Studio Code 官网下载最新版本的 VS Code 安装包，并在安装时勾选 添加到 PATH 。

    同时，建议勾选其他三项

  • 添加自动打开编辑器脚本
    在 博客根目录/scripts/ 下新建 AutoOpenEditor.js 文件（如果没有 scripts 目录则新建），并粘贴以下代码，保存。

这样，在每次 hexo new 的时候，脚本就会自动打开 VS Code 并切换到博客根目录顺带打开新建的 .md 文件
