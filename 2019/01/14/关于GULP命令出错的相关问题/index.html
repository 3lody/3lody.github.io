npm安装gulp后gitbash不识别命令

$ gulp bash: gulp: command not found

解决方法:

  1. 打开路径：C:\Users\Administrator下的.nmprc文件 修改prefix=C:\Program Files\nodejs\node_modules\npm（命令where node查文件路径）

  2. 在项目文件夹中，全局安装gulp

  3. 安装开发路径的gulp

  4. 添加电脑环境变量-编辑用户PATH

运行gulp项目报错

运行gulp项目报错：AssertionError: Task function must be specified

解决方法：

  1. gulp项目需要全局安装gulp和项目内安装gulp，通过 gulp -v 查看全局gulp 和本地项目的gulp版本
  2. 可以卸载修改json文件里面的版本号：为gulp 3 版本。如 3.9.1
  • 修改hexo根目录下的package.json

  1. 重新运行 npm install

  2. gulp 3 和 gulp 4运行方式是不一样的，所以会出现错误
